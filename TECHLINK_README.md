## ğŸ¯ How It Works

### Freelancer Journey
1. Register with skills, education, and hourly rate
2. Browse opportunities (budget hidden from view)
3. Apply to jobs with cover letter and estimated hours
4. Receive offers from clients
5. Admin reviews and approves after assessment
6. Contract created and work begins
7. Log hours worked throughout project
8. Mark project complete when done
9. Receive payment based on hours worked

### Client Journey
1. Register as individual or company
2. Post job with description, timeline, and budget
3. Browse available freelancers
4. Review applications from interested freelancers
5. Send offers to selected candidates
6. Admin approves and creates contract
7. Monitor project progress
8. Receive invoice based on hours worked

### Admin Workflow
1. Monitor all platform activity
2. Review applications in interview stage
3. Check freelancer availability
4. Approve contracts (prevents double-booking)
5. Track project progress and hours
6. Process payments when projects complete
7. Generate invoices for clients
8. Earn 25% commission on all projects
